TrueNAS MCP Server

Research Preview Available

A Model Context Protocol (MCP) server for TrueNAS that enables AI assistants like Claude to manage TrueNAS systems through natural language. The server runs as a native binary on your desktop and securely connects to your TrueNAS system without requiring any deployment to TrueNAS itself.

Key Features:

  • System monitoring and health checks
  • Storage pool, dataset, and snapshot management
  • Virtual machine management
  • Directory service integration (AD, LDAP, FreeIPA)
  • Capacity planning and utilization analysis
  • Application management with task tracking
  • Dry-run mode and built-in safety validations
